## 🌟 ¿Qué es este curso?

Este es un curso **completamente gratuito** de Python diseñado para personas que **nunca han programado**. No necesitas saber nada de computación. Solo necesitas una computadora y ganas de aprender.

> 💡 **Programar es como dar instrucciones a una computadora.** En este curso aprenderás a hacerlo paso a paso, como si estuvieras aprendiendo a cocinar siguiendo una receta.

## 🚀 ¿Cómo empezar? (Versión ultra rápida)

### Paso 1 — Descarga el curso
```
Haz clic en el botón verde "Code" ↑ → "Download ZIP" → Extrae la carpeta
```

### Paso 2 — Instala Python
```
Ve a python.org → Downloads → Descarga → Instala
⚠️ MUY IMPORTANTE: Marca "Add Python to PATH" durante la instalación
```

### Paso 3 — Instala VS Code
```
Ve a code.visualstudio.com → Descarga → Instala
```

### Paso 4 — Abre el primer archivo
```
En VS Code: Archivo → Abrir carpeta → Selecciona la carpeta del curso
Abre: Fase1_Fundamentos/dia01_hola_mundo.py
Haz clic en ▶️ (arriba a la derecha)
```

> 📖 **¿Necesitas más ayuda?** Lee el [MANUAL_COMPLETO.md](MANUAL_COMPLETO.md) donde todo está explicado con capturas y ejemplos.
